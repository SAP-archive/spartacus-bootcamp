Static page

Pages can be created in spartacus in a generic way. Spartacus let you work with the router in a generic way.

Standard router

The standard angular router works as expected. You can use the routerLink to link to any pages, for example the home page.

Semantic Links

This section demonstrate semantic links in Spartacus, for example a link to the cart page. Using the cxUrl, you can use a semantic link which is configurable and controlled in Spartacus.
